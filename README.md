📊 Regional Sales Analytics Dashboard
Project Overview

Developed sales analytics solution using Python and Power BI to analyze sales performance, identify business trends, and support data-driven decision-making.

**Objectives**
Analyze sales trends and revenue performance
Identify top-performing products and regions
Track key business KPIs
Create interactive dashboards for stakeholders

**Tools Used**
Python (Pandas, NumPy)
Power BI
Excel/CSV

**Key Features**
Data cleaning and transformation
KPI monitoring
Regional sales analysis
Product performance analysis
Interactive filters and slicers

**Key Insights**
Identified top-performing products contributing the highest revenue.
Analyzed regional sales distribution and growth opportunities.
Tracked monthly sales trends and performance metrics.
Improved reporting efficiency through interactive visualizations.

**Skills Demonstrated**
Data Cleaning
Data Transformation
Exploratory Data Analysis (EDA)
Data Visualization
Dashboard Development
Business Intelligence
